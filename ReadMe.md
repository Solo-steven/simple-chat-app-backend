## Simple Demo app

Using Socket.io, React, Redux,and Express with MongoDB

frontend repo: https://github.com/Solo-steven/simple-chat-app-frontend

backend repo: https://github.com/Solo-steven/simple-chat-app-backend

## Setup

- backend

```
npm run db:start
npm run serve
```

- frontend

```
yarn start
```
